# Política de Privacidad - UNAM Ciencias Personalizador Visual

## Compromiso con la Privacidad

Esta extensión ha sido diseñada con la privacidad y seguridad como prioridades absolutas, especialmente considerando que se usa en un portal de servicios estudiantiles que maneja información académica sensible.

## Información que NO Recopilamos

✅ **Datos personales**: Nombres, números de cuenta, información académica
✅ **Credenciales**: Contraseñas, tokens de autenticación  
✅ **Actividad de navegación**: Páginas visitadas, tiempo de uso
✅ **Información del formulario**: Datos ingresados en formularios del sitio
✅ **Información del dispositivo**: Dirección IP, ubicación, hardware

## Información que SÍ Almacenamos (Solo Localmente)

La extensión guarda únicamente en tu navegador local:
- ✓ Estado activado/desactivado de la extensión
- ✓ Preferencias de tema visual seleccionado
- ✓ Configuraciones de apariencia personalizadas

**Importante**: Estos datos nunca salen de tu navegador y se eliminan si desinstalas la extensión.

## Permisos de la Extensión

### `activeTab`
- **Propósito**: Aplicar estilos CSS únicamente en la pestaña activa
- **Limitación**: Solo funciona cuando visitas web.fciencias.unam.mx
- **No permite**: Acceso a otras pestañas o sitios web

### `storage`
- **Propósito**: Guardar preferencias de tema localmente
- **Limitación**: Solo datos de configuración visual
- **No permite**: Acceso a datos del sitio web original

### `host_permissions` para web.fciencias.unam.mx
- **Propósito**: Inyectar estilos CSS solamente en el sitio objetivo
- **Limitación**: Exclusivamente el dominio de Ciencias UNAM
- **No permite**: Acceso a otros sitios web

## Funcionamiento Técnico Seguro

### Solo CSS y JavaScript Local
- La extensión únicamente modifica la apariencia visual
- No intercepta formularios ni datos del usuario
- No modifica la funcionalidad original del sitio
- Funciona completamente offline una vez instalada

### Sin Conexiones Externas
- No envía datos a servidores remotos
- No descarga contenido de internet
- No utiliza servicios de terceros
- No incluye sistemas de telemetría o analytics

### Código Fuente Abierto
- Todo el código está disponible para revisión
- Sin código ofuscado o minificado
- Auditable por la comunidad universitaria

## Seguridad en Trámites Estudiantiles

### Protección de Credenciales
- La extensión NO accede a campos de contraseñas
- NO lee ni modifica información de formularios
- NO interfiere con procesos de autenticación
- Solo cambia colores, fuentes y espaciado visual

### Integridad de Trámites
- Todos los procesos estudiantiles funcionan normalmente
- Sin impacto en constancias, inscripciones o pagos
- La funcionalidad original se mantiene intacta
- Posibilidad de desactivar instantáneamente si es necesario

## Transparencia y Control del Usuario

### Control Total
- Activación/desactivación instantánea desde el popup
- Restablecimiento de tema original en un clic
- Desinstalación completa sin rastros de datos

### Información Clara
